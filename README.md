# Telegram Stars Shop Bot 🤖⭐

Бот для продажи Telegram Stars через криптовалюту (USDT) с использованием Crypto Bot. Написан на Python с использованием aiogram и SQLite.

## ✨ Возможности

- **Покупка Telegram Stars** за USDT (через Crypto Bot)
- **Inline-клавиатуры** для удобной навигации
- **Система платежей** с проверкой статуса
- **Личный кабинет** с историей покупок
- **Топ покупателей** (рейтинг)
- **Админ-панель** со статистикой
- **База данных SQLite** для хранения пользователей и платежей
- Получение звёзд НЕ реализовано

## 🛠 Технологии

- Python 3.9+
- aiogram 3.x
- SQLite
- Crypto Bot API
- python-dotenv

## 📦 Установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/yourusername/telegram-stars-shop-bot.git
cd telegram-stars-shop-bot
```
2. **Замените данные в конфиге(config.py)**
3. **Запустите бота**
```bash
python main.py
```
